## מהם סלוטים אונליין?

סלוטים אונליין הם הגרסה הדיגיטלית של מכונות המזל הקלאסיות. במקום למשוך ידית — לוחצים על כפתור. גלילים מסתובבים, סמלים נוחתים, ואם נוצר צירוף זוכה — מקבלים תשלום. הכל אוטומטי, מהיר ומרגש.

**מונחים חשובים:**

| מונח | הסבר |
|-------|------|
| **RTP (Return to Player)** | אחוז ההחזר התיאורטי. RTP 96% = על כל ₪100 שמשחקים, מוחזרים ₪96 בממוצע |
| **Volatility (תנודתיות)** | גבוהה = זכיות גדולות אבל נדירות. נמוכה = זכיות קטנות אבל תכופות |
| **Paylines (קווי תשלום)** | הצירופים שמשלמים. 10-100+ קווים |
| **Wild** | סמל שמחליף כל סמל אחר (כמו ג'וקר) |
| **Scatter** | סמל שמפעיל פיצ'ר מיוחד (ספינים חינם, בונוס) |
| **Free Spins** | סיבובים חינם — ללא עלות, עם זכיות אמיתיות |
| **Multiplier** | מכפיל שמגדיל את הזכייה (x2, x5, x100...) |
| **Progressive Jackpot** | ג'קפוט שגדל כל שנייה — כל הימור מוסיף לקופה |

### מהו RTP וכמה צריך לחפש?

### מה ההבדל בין תנודתיות גבוהה לנמוכה?
תנודתיות גבוהה = זכיות גדולות אבל נדירות (צריך סבלנות ובנקרול). תנודתיות נמוכה = זכיות קטנות אבל תכופות (מתאים לתקציב מוגבל ומשחק ממושך).

כן, ברוב המקרים. ברבים מהסלוטים הפרוגרסיביים כל הימור זכאי לג'קפוט. אבל: ככל שההימור גדול יותר, הסיכוי לג'קפוט גדל. במשחקים מסוימים צריך הימור מקסימלי לג'קפוט הגדול.

### מה עדיף — סלוטים או רולטה?
